# 🍷 Wine Quality Data Analysis

This project focuses on analysing the physicochemical properties of wine and understanding how they influence its quality.

---

## Objective

The main goal of this project is to:

- Analyse the relationship between wine features and quality  
- Identify patterns and trends in the dataset  
- Support data-driven insights about wine characteristics  

---

## 📊 Dataset

The dataset used is **WineQT.csv**, which contains physicochemical attributes of wine samples.

### Features:
- Fixed acidity  
- Volatile acidity  
- Citric acid  
- Residual sugar  
- Chlorides  
- Free sulfur dioxide  
- Total sulfur dioxide  
- Density  
- pH  
- Sulphates  
- Alcohol  

### Target:
- `quality` → wine quality score
